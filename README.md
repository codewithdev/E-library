### Requirement to run the Project on Local Server

 - XAMPP 
 - Code Editor(Sublime, VS Code etc)
 
 - Clone the Repository from `git@github.com:codewithdev/E-library.git` on your local machine in the `htdocs` folder of xampp.
 - Start the Server
